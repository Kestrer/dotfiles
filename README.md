# my dotfiles

Just run `make install` to install to your system. Remember to delete your old configs first otherwise it won't install; to do this run `make uninstall`.

## Required Software:

- dbus
- dunst
- feh
- fontawesome
- i3status
- i3wm
- make
- neovim
- notify-send
- pcre2grep
- qutebrowser
- rofi
- rofi-calc
- scrot
- slock (with dpms patch)
- st (with scrollback patch)
- sxhkd
- TeX
- xbanish
- xclip
- xorg
- zathura
- zsh
